特性

核心功能

01

多平台视频支持

支持 YouTube、Bilibili、Twitter/X、Vimeo 等主流视频平台，自动检测并下载可用的字幕内容。

使用 yt-dlp 作为后端，支持手动字幕和自动生成字幕，优先选择质量更高的手动字幕。

对于 B站视频，会自动检测 CC 字幕；对于 YouTube 视频，支持多语言字幕选择。

02

智能翻译与整理

英文字幕自动翻译为流畅的中文，并根据内容自动整理为结构清晰的文章格式。

翻译不仅仅是字面转换，而是会根据上下文语境进行调整，使文章更加通顺易读。

03

精美排版设计

提供多种文章风格选择：经典优雅、Claude风格、极简现代、杂志风格，每种风格都经过精心设计。

所有生成的 HTML 都支持响应式布局，在手机、平板、电脑上都能获得良好的阅读体验。

开始使用

安装 yt-dlp，然后在 Claude Code 中输入视频链接，即可自动生成精美文章。
