# 公众号故事叙述文体模板

> 版本 1.0 · 适用于案例分享、经历讲述、人物故事

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的内容基于视频字幕整理加工
- 公众号发布前请务必添加原始视频来源信息
- 建议在文章开头或结尾注明「内容整理自视频：标题」
- 请尊重原创，可添加原文链接和原频道信息
- 如需商业使用，建议联系原视频作者获得授权

## 必加来源信息格式

```markdown
---
来源说明：
本文整理自 YouTube 视频《视频标题》
原视频链接：https://www.youtube.com/watch?v=xxx
原频道：@频道名称
---
```

## 文体特征

| 维度 | 特征 |
|------|------|
| **字数范围** | 1500-2500 字 |
| **标题规范** | 故事关键词 + 情感钩子 |
| **排版要求** | 故事线清晰，高潮突出 |
| **视觉元素** | 场景描写、对话引用 |
| **阅读体验** | 引人入胜、有代入感 |
| **互动引导** | 情感共鸣式结尾 |

## 结构模板

```
【标题】故事关键词 + 情感钩子

开头：场景切入 + 人物引入

发展：故事主体

高潮：转折/顿悟

结尾：升华 + 引发思考
```

## 生成提示词

```
你是一位资深公众号作者，擅长讲述打动人心的故事。

请将以下视频内容转化为公众号故事文，要求：

1. 文体类型：故事叙述文
2. 标题：故事关键词 + 情感钩子
3. 正文：1500-2500字
4. 格式要求：
   - 使用 Markdown 格式
   - 有清晰的故事线（起承转合）
   - 场景描写生动
   - 适当使用对话/引用
   - 结尾升华主题，引发思考
5. 风格：故事性强、有共鸣、打动人
6. 输出格式：Markdown，保留格式符号

视频内容摘要：
{content}
```
