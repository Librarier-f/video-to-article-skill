# 公众号金句提炼文体模板

> 版本 1.0 · 适用于内容精华、语录整理、灵感记录

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的内容基于视频字幕整理加工
- 公众号发布前请务必添加原始视频来源信息
- 建议在文章开头或结尾注明「内容整理自视频：标题」
- 请尊重原创，可添加原文链接和原频道信息
- 如需商业使用，建议联系原视频作者获得授权

## 必加来源信息格式

```markdown
---
来源说明：
本文整理自 YouTube 视频《视频标题》
原视频链接：https://www.youtube.com/watch?v=xxx
原频道：@频道名称
---
```

## 文体特征

| 维度 | 特征 |
|------|------|
| **字数范围** | 1000-1500 字 |
| **标题规范** | 来源 + 「精华/金句」 |
| **排版要求** | 金句突出，解读简洁 |
| **视觉元素** | 引用块、分割线 |
| **阅读体验** | 轻松、易消化 |
| **互动引导** | 分享式结尾 |

## 结构模板

```
【标题】来源 + 「精华/金句」

开头：背景介绍

正文：金句清单 + 简要解读

结尾：总结 + 一句话升华
```

## 示例输出

```markdown
---
来源说明：
本文整理自 YouTube 视频《6个吸引力法则》
原视频链接：https://www.youtube.com/watch?v=TBxS0XhdfmU
原频道：改善生活频道
---

# 这6句话，治好了我的社交焦虑

最近看了一个研究「吸引力」的视频，全程45分钟，我记下了最打动我的6句话。

每一句，都像是对我之前社交困境的回答。

---

**1. 「与他人的任何连接，都必须从你自己开始。」**

我们总想着如何表现才能被喜欢。

但真正的连接，是先与自己同在——真正地存在于自己的身体里，放松、从容、不匆忙。

---

**2. 「管理他人的情绪不是你的责任。」**

社恐人最怕的就是冷场。

但视频里说：别急着填补沉默。沉默不是你的错。

---

**3. 「一个人的名字是任何语言中最甜美的声音。」**

记住并使用对方的名字。

这个小动作，传递的是「你很重要」。

---

**4. 「这个人对我的意图是好的吗？」**

这是每个人见到陌生人时无意识问的问题。

所以，先给温暖。假定对方喜欢你，你先迈出第一步。

---

**5. 「有吸引力的人是能量的给予者，不是索取者。」**

当有人进入房间时，人们会评估：这个人是让我的能量上升还是下降？

成为前者。

---

**6. 「磁力不是你有多漂亮，而是你能让别人在你面前感觉如何。」**

这是整个视频最核心的一句话。

吸引力不是关于你，而是关于你让对方感觉如何。

---

**写在最后**

这些话，我会打印出来贴在工位上。

每次社交前，看一眼，提醒自己：

不用表演，不用紧张。

真正地「在」那里，就够了。

---

*觉得有用的话，点个「在看」吧 ❤️*
```
