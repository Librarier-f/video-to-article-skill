# 公众号深度长文文体模板

> 版本 1.0 · 适用于知识科普、深度分析、方法论

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的内容基于视频字幕整理加工
- 公众号发布前请务必添加原始视频来源信息
- 建议在文章开头或结尾注明「内容整理自视频：标题」
- 请尊重原创，可添加原文链接和原频道信息
- 如需商业使用，建议联系原视频作者获得授权

## 必加来源信息格式

```markdown
---
来源说明：
本文整理自 YouTube 视频《视频标题》
原视频链接：https://www.youtube.com/watch?v=xxx
原频道：@频道名称
---
```

## 文体特征

| 维度 | 特征 |
|------|------|
| **字数范围** | 2000-3000 字 |
| **标题规范** | 主题 + 增量价值 |
| **排版要求** | 3-5个核心模块 |
| **视觉元素** | 章节编号、要点列表 |
| **阅读体验** | 系统性、完整性 |
| **互动引导** | 文末行动建议 |

## 结构模板

```
【标题】主题 + 增量价值

开头：痛点引入 + 文章价值预告

正文：3-5个核心模块
每个模块：概念 + 案例 + 方法

结尾：总结 + 行动建议 + 关注引导
```

## 生成提示词

```
你是一位资深公众号作者，擅长撰写有深度的系统性长文。

请将以下视频内容转化为公众号文章，要求：

1. 文体类型：深度长文
2. 标题：主题 + 价值增量
3. 正文：2000-3000字
4. 格式要求：
   - 使用 Markdown 格式
   - 3-5个清晰的核心模块
   - 每个模块包含：概念讲解 + 案例 + 实践方法
   - 文末添加行动建议和关注引导
5. 风格：系统性、有深度、可操作
6. 输出格式：Markdown，保留格式符号

视频内容摘要：
{content}
```
