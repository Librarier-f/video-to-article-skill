# 情绪共鸣型口播稿模板

> 版本 1.0 · 适用于情感话题、人生感悟、励志鸡汤、痛点共鸣

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的口播稿基于视频字幕整理加工
- 用于配音拍摄前请务必添加原始视频来源信息
- 建议在口播稿开头或视频描述中注明来源
- 请尊重原创，可注明「整理自@频道名称」
- 如视频平台有原创标识要求，请按平台规则处理

## 必加来源信息格式（口播稿开头）

```
【内容来源】
本视频内容整理自：《视频标题》
原视频：https://www.youtube.com/watch?v=xxx
原频道：@频道名称

【正文开始】
```

## 适用场景
- 情感话题
- 人生感悟
- 励志鸡汤
- 痛点共鸣

## 核心特点
精准戳中观众痛点，让他们觉得"说的就是我"，然后给出希望

## 结构模板

```
【共情开场 - 8秒】
你是不是也这样？
每天早上XX，闹钟XX，结果还是XX。
明明知道XX，但又忍不住XX。
（停顿）
我也曾经这样。

【痛点放大 - 12秒】
那种感觉真的很糟糕，对吧？
每天XX，XX，结果什么都没做成。
然后开始XX，晚上又XX。
你是不是也这样循环过？
（停顿）
我也懂，真的。

【转折希望 - 15秒】
但后来我发现，问题不是我们不够努力。
而是我们XX。
听起来有点反直觉，对吧？
但当你XX，你会发现XX。
原来XX。

【力量给予 - 10秒】
从今天开始，你可以试试这个方法。
不需要XX，只需要XX。
每天花X分钟，坚持一周。
你会发现XX。

【行动号召 - 5秒】
如果你也曾经这样，评论区告诉我。
觉得有收获的话，点赞。
我是XX，我们下期见。
```

## 关键要素

| 要素 | 要求 |
|------|------|
| 共情句式 | "你是不是也这样"、"我懂"、"那种感觉" |
| 痛点具体化 | 细节越具体越有代入感 |
| 转折点 | "但后来"、"但其实"后给希望 |
| 停顿标注 | 情绪高点用"（停顿）"标注 |
| 正能量 | 结尾要有力量感，不要丧 |

## 示例开场白

```
"你是不是每天早上闹钟响了三遍，还是不想起床？
明明知道早起好，但就是做不到。
那种感觉真的很糟糕，对吧？
我也曾经这样。"
```

## 情绪词使用技巧

```
✅ 有效共情：
"你有没有过这种感觉，忙了一天，但晚上躺下来却觉得什么都没做。
明明很累，但又睡不着。
第二天起来更累。
——你是不是也这样？"

❌ 无效共情：
"人要自律，不能懒，要努力奋斗。"
（太说教，没有代入感）
```

## 痛点到希望的转折技巧

```
❌ 生硬转折：
"但是！我们不能这样下去！要努力！"

✅ 自然转折：
"但后来我想通了，问题不在于意志力不够。
而在于我们的方法从一开始就不对。
当你用对方法之后，一切都会变得不一样。"
```

## 适用话题特征
- ✅ 适合：早起、拖延、焦虑、自我怀疑、职场倦怠
- ❌ 不适合：硬核知识、客观事实、争议话题

## 注意事项
- 痛点要真实存在，不能编造
- 给出希望时要有具体方法，不能只喊口号
- 情绪要有起伏，不能全程低沉
- 力量感要真实，不要假大空
- 多用"我"、"你"，少用"大家"、"人们"
