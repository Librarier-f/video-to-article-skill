# 故事叙述型口播稿模板

> 版本 1.0 · 适用于个人经历分享、经验复盘、创业/职场故事

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的口播稿基于视频字幕整理加工
- 用于配音拍摄前请务必添加原始视频来源信息
- 建议在口播稿开头或视频描述中注明来源
- 请尊重原创，可注明「整理自@频道名称」
- 如视频平台有原创标识要求，请按平台规则处理

## 必加来源信息格式（口播稿开头）

```
【内容来源】
本视频内容整理自：《视频标题》
原视频：https://www.youtube.com/watch?v=xxx
原频道：@频道名称

【正文开始】
```

## 适用场景
- 个人经历分享
- 经验复盘
- 创业/职场故事

## 核心特点
用真实故事吸引观众，通过情节起伏传递价值，让人有代入感

## 结构模板

```
【开场钩子 - 5秒】
三年前，我XX。
那时候我以为XX，结果现实告诉我XX。
如果你也XX，这条视频可能会帮到你。

【背景铺垫 - 10秒】
那时候我每天XX，闹钟XX，XX。
每天的状态就是XX。
说实话，XX。
我也试过XX，但都XX。

【冲突爆发 - 15秒】
直到有一天，发生了XX。
那天的具体情况是XX。
所有人都XX，包括我自己也觉得XX。
但后来我发现，XX其实不是XX。

【转折发现 - 15秒】
后来我遇到了XX，学到了XX。
原来，XX才是关键。
当我开始XX，我才发现XX。
这种感觉，就像XX。

【启发总结 - 10秒】
这段经历让我明白了一个道理：
XX不是XX，而是XX。
与其XX，不如XX。
这改变了我后来的XX。

【行动号召 - 5秒】
如果你正在经历类似的阶段，评论区聊聊。
觉得有收获的话，点赞告诉我。
我是XX，我们下期见。
```

## 关键要素

| 要素 | 要求 |
|------|------|
| 时间锚点 | "三年前"、"有一次"、"去年"建立真实感 |
| 情节起伏 | 困难→转折→结果，有弧线 |
| 感官细节 | 加入具体场景、画面感 |
| 对话感 | 用"我记得"、"那天"、"然后"推进 |
| 金句提炼 | 结尾必须有1句核心感悟 |

## 示例开场白

```
"三年前，我每天早上闹钟响三遍都起不来。
那时候我以为是自己太懒，后来才发现，问题根本不在这里。
如果你也总是起床困难，这条视频可能会帮到你。"
```

## 故事推进技巧

```
❌ 平铺直叙：
"我以前起不来床，后来我找到了方法。"

✅ 有起伏感：
"我试过把闹钟放远，试过早睡，试过各种方法。
但都没用。
直到有一天，我看到一个研究说，起不来的原因不是懒，而是XX。
那一刻我突然明白了。"
```

## 适用话题特征
- ✅ 适合：成长经历、职场经验、学习方法、自我提升
- ❌ 不适合：纯知识科普、硬核干货、时事评论

## 注意事项
- 故事要有"代入感"，让观众想到自己的经历
- 转折点要出乎意料但又合乎逻辑
- 感悟要具体，不要太空洞
- 控制时长，整体不超过90秒
- 避免太个人化的细节，要找到与观众的共鸣点
