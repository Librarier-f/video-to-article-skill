# 清单列表型口播稿模板

> 版本 1.0 · 适用于技巧盘点、方法汇总、步骤讲解、清单合集

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的口播稿基于视频字幕整理加工
- 用于配音拍摄前请务必添加原始视频来源信息
- 建议在口播稿开头或视频描述中注明来源
- 请尊重原创，可注明「整理自@频道名称」
- 如视频平台有原创标识要求，请按平台规则处理

## 必加来源信息格式（口播稿开头）

```
【内容来源】
本视频内容整理自：《视频标题》
原视频：https://www.youtube.com/watch?v=xxx
原频道：@频道名称

【正文开始】
```

## 适用场景
- 技巧盘点
- 方法汇总
- 步骤讲解
- 清单合集

## 核心特点
用数字锚定节奏，逐条展开，逻辑清晰，听众记得住

## 结构模板

```
【开场 - 5秒】
今天给你分享X个XX的方法，都是我亲自验证过的。
特别适合总是XX的你。

【总起 - 5秒】
这X点特别关键，关系到XX，建议先收藏起来。

【第一点 - 15秒】
第一点，XX。
（停顿1秒）
说到这个，XX（展开说明/个人体验/具体例子）。
【过渡】其实XX，但最重要还是这一点。

【第二点 - 15秒】
第二点，XX。
你有没有发现，XX（痛点描述）？
对吧，这个问题很普遍。
其实解决起来很简单，XX（解决方法）。
【过渡】好，我们接着说第三点。

【第三点 - 15秒】
第三点，XX。
这可能是最容易被忽略的一条。
很多人XX，结果XX（反面案例）。
但如果你XX，就会发现XX（正面效果）。

【总结召唤 - 5秒】
好了，以上就是今天的X点分享。
都挺实用的，特别是第X点，坚持一周你就会看到变化。
觉得有用的话，点赞告诉我，我们下期见。
```

## 关键要素

| 要素 | 要求 |
|------|------|
| 数字选择 | 3、5、7最佳，记忆负担小 |
| 对话感 | 多用"你有没有发现"、"对吧"、"其实" |
| 连续性 | 每点之间用"好，接着说"、"下面"、"然后"连接 |
| 停顿标记 | 关键句后停顿0.5-1秒，用"（停顿）"标注 |

## 示例开场白

```
"今天给你分享3个早起后立刻清醒的方法，都是我亲身试过的。
特别适合每天闹钟响了三遍还是不想动的你。"
```

## 列表项写法技巧

```
❌ 太抽象：
"第一，要保持好心态"

✅ 对话版：
"第一，把闹钟放远一点。
你有没有过这种经历，躺着一伸手就关掉闹钟，结果又睡过去了？
把闹钟放到必须下床才能碰到的地方，这是最简单的一步，但真的有效。"
```

## 适用话题特征
- ✅ 适合：方法论、步骤教程、技巧盘点、习惯养成
- ❌ 不适合：纯情感倾诉、需要深层理解的内容

## 注意事项
- 列表项数量控制在3-5个最佳
- 每点要有独立价值，不重复
- 避免太学术的词汇，用口语化表达
- 结尾要有行动召唤，不要戛然而止
