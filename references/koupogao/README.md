# 口播稿模板说明

口播稿是专门为短视频配音设计的文案类型，具有强节奏感、高共鸣、纯口语化的特点。

## 五种口播稿类型

| 类型 | 文件 | 适用场景 |
|------|------|---------|
| 爆款开场型 | `baokuan.md` | 知识干货、认知颠覆 |
| 故事叙述型 | `gushi.md` | 个人经历、经验分享 |
| 清单列表型 | `qingdan.md` | 技巧盘点、方法汇总 |
| 对话挑战型 | `duihua.md` | 观点表达、争议话题 |
| 情绪共鸣型 | `qingxu.md` | 情感话题、人生感悟 |

## 核心特点

### 对话性
- 多用"你"、"我"，少用"大家"、"人们"
- 用"你有没有发现"、"对吧"、"其实"拉近距离
- 像朋友聊天，不要像老师上课

### 连续性
- 段落间用连接词：然后、接着、下面、所以、因为
- 避免跳跃感，每句话都能自然过渡到下一句
- 用"（停顿）"标注情绪高点或需要换气的地方

### 直接可用
- 完整的开场、结尾，不用再加工
- 语气词、停顿位置都已标注
- 拿到就能读，读起来顺口

## 核心结构

```
【开场钩子】0-5秒    → 反问/冲突/共情，拉住注意力
【过渡连接】5-10秒   → 建立期待，"这条视频一定要看完"
【核心内容】10-60秒  → 2-3个要点，每点有展开
【总结召唤】最后5秒  → 点赞/评论引导
【固定结尾】          → "我是XX，我们下期见"
```

## 输出格式

| 格式 | 说明 |
|------|------|
| `.txt` | 纯文本口播稿，可直接用于配音 |
| `-预览.html` | 手机屏幕预览效果 |

## 使用建议

1. **先选类型**：根据内容主题选择最合适的口播稿类型
2. **替换内容**：把模板中的XX换成具体内容
3. **调整节奏**：根据语速调整每段的时长
4. **预览确认**：用HTML预览文件在手机上查看效果
5. **添加来源**：口播稿末尾请添加内容来源说明

## 模板使用示例

```
原始模板句式：
"你是不是也这样？每天早上XX，闹钟XX，结果还是XX。"

替换后：
"你是不是也这样？每天早上闹钟响了三遍，还是不想起床，
明明知道早起好，但就是做不到。"
```
