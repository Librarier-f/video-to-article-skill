# 小红书情感共鸣文体模板

> 版本 1.0 · 适用于情感故事、人生感悟、治愈系内容

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的内容基于视频字幕整理加工
- 发布前请务必添加原始视频来源信息
- 建议注明「内容整理自@原频道名」或「素材来源：视频标题」
- 请尊重原创，仅进行二次加工而非抄袭

## 必加来源信息格式

```
📺 内容来源：视频标题
🔗 原视频链接：https://www.youtube.com/watch?v=xxx
📢 转载/整理自 @频道名称
```

## 文体特征

| 维度 | 特征 |
|------|------|
| **字数限制** | 正文建议 300-500 字 |
| **标题规范** | 情绪触发 + 场景/话题 |
| **段落风格** | 短句、诗意的断行 |
| **视觉元素** | 情感emoji为主 |
| **互动引导** | 分享式互动 |
| **标签系统** | 底部添加情感类标签 |

## 结构模板

```
【标题】emoji + 情绪触发 + 场景/话题

【正文】
（情绪引入，制造共鸣）
……

💭 想法/感悟
……

❤️ 治愈/鼓励
……

🔄 互动话术

【标签】
```

## 示例输出

⚠️ 注意：以下示例为纯文本格式，禁止包含任何 Markdown 符号

```
📺 内容来源：6个吸引力法则
🔗 原视频：https://www.youtube.com/watch?v=TBxS0XhdfmU
📢 整理自改善生活频道
━━━━━━━━━━━━━━━━━━━━━━━━

💕 原来吸引力，是可以练习的

一直以为吸引力是天生的
直到看到这个视频，突然释怀了……

💭 想起以前总是小心翼翼
害怕说错话、害怕冷场
拼命想表现得好
结果越努力越尴尬

原来，真正有吸引力的人
不是努力表现
而是——真正地「在」那里

❤️ 视频里有句话很戳我：
当你让他们感到被看见和充满活力时
他们自然会想要在你身边

原来不是要变得多完美
而是让别人在你面前感觉更好

🔄 分享给同样在努力的你
愿我们都能成为让人想要靠近的人

#治愈 #自我成长 #人生感悟
```
