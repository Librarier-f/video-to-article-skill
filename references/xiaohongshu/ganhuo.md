# 小红书干货教程文体模板

> 版本 1.0 · 适用于技能分享、方法教程、知识科普

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的内容基于视频字幕整理加工
- 发布前请务必添加原始视频来源信息
- 建议注明「内容整理自@原频道名」或「素材来源：视频标题」
- 请尊重原创，仅进行二次加工而非抄袭

## 必加来源信息格式

```
📺 内容来源：视频标题
🔗 原视频链接：https://www.youtube.com/watch?v=xxx
📢 转载/整理自 @频道名称
```

## 文体特征

| 维度 | 特征 |
|------|------|
| **字数限制** | 正文建议 300-600 字 |
| **标题规范** | 带数字，更吸睛 |
| **段落风格** | 步骤清晰，分段明确 |
| **视觉元素** | 序号emoji、要点符号 |
| **互动引导** | 结尾引导评论 |
| **标签系统** | 底部添加 3-5 个 #话题标签 |

## 结构模板

```
【标题】emoji + 数字 + 方法/技巧 + 收藏提醒

【正文】
⚠️ 建议收藏！……

📚 背景/原理（简短引入）
……

✅ 方法/步骤（核心内容）
🔸 第一步：……
🔸 第二步：……
🔸 第三步：……

💡 关键技巧/注意事项
……

📌 总结回顾
……

💬 互动话术

【标签】
```
