# 小红书清单盘点文体模板

> 版本 1.0 · 适用于资源整理、清单合集、推荐汇总

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的内容基于视频字幕整理加工
- 发布前请务必添加原始视频来源信息
- 建议注明「内容整理自@原频道名」或「素材来源：视频标题」
- 请尊重原创，仅进行二次加工而非抄袭

## 必加来源信息格式

```
📺 内容来源：视频标题
🔗 原视频链接：https://www.youtube.com/watch?v=xxx
📢 转载/整理自 @频道名称
```

## 文体特征

| 维度 | 特征 |
|------|------|
| **字数限制** | 正文建议 400-600 字 |
| **标题规范** | emoji + 数字 + 主题 |
| **段落风格** | 清单式，一目了然 |
| **视觉元素** | 序号emoji、标题加粗感 |
| **互动引导** | 提问式互动 |
| **标签系统** | 底部添加主题标签 |

## 结构模板

```
【标题】emoji + 数字 + 主题 + 收藏提醒

【正文】
整理了……，建议收藏！
👇👇👇

1️⃣ 【标题】
内容……

2️⃣ 【标题】
内容……

（以此类推）

📌 收藏起来慢慢看！

💬 你最喜欢哪个？

【标签】
```

## 生成提示词

```
你是一位资深小红书博主，擅长撰写高收藏价值的清单类笔记。

请将以下视频内容转化为小红书笔记，要求：

1. 文体类型：清单盘点文
2. 标题：emoji + 数字 + 主题，20字以内
3. 正文：400-600字
4. 格式要求：
   - 使用 1️⃣ 2️⃣ 3️⃣ 作为序号
   - 每个要点有简短标题
   - 要点内容1-3行
   - 结尾添加收藏提示
   - 底部添加3-5个#话题标签
5. 风格：干货满满、收藏价值高
6. 输出格式：纯文本，不使用任何 Markdown 语法

视频内容摘要：
{content}
```
