# 小红书种草安利文体模板

> 版本 1.0 · 适用于产品推荐、好物分享、课程安利

## ⚠️ 内容来源与使用声明

**重要提示：**
- 本模板生成的内容基于视频字幕整理加工
- 发布前请务必添加原始视频来源信息
- 建议注明「内容整理自@原频道名」或「素材来源：视频标题」
- 请尊重原创，仅进行二次加工而非抄袭

## 必加来源信息格式

```
📺 内容来源：视频标题
🔗 原视频链接：https://www.youtube.com/watch?v=xxx
📢 转载/整理自 @频道名称
```

## 文体特征

| 维度 | 特征 |
|------|------|
| **字数限制** | 正文建议 300-600 字 |
| **标题规范** | 20 字以内，带 emoji |
| **段落风格** | 短句为主，多用换行 |
| **视觉元素** | Emoji 装饰、符号列表 |
| **互动引导** | 结尾必带互动话术 |
| **标签系统** | 底部添加 3-5 个 #话题标签 |

## 结构模板

```
【标题】emoji + 痛点/效果 + 产品名

【正文】
姐妹们！今天必须安利一个……

✨ 为什么推荐？（痛点共鸣）
• 痛点1：……
• 痛点2：……

💝 使用体验（核心内容）
• 体验点1：……
• 体验点2：……

📝 总结（干货提炼）
• 要点1
• 要点2
• 要点3

💬 互动话术
姐妹们还有什么想了解的？评论区见！

【标签】#标签1 #标签2 #标签3
```

## 标题生成规则

| 规则 | 示例 |
|------|------|
| 数字开头 | `6个技巧让你……` |
| 疑问句式 | `为什么有些人……？` |
| 对比反差 | `普通人vs高情商……` |
| 情绪触发 | `看完哭了……` |
| 收藏暗示 | `建议收藏！……` |

## Emoji 使用指南

| 类型 | 推荐 Emoji |
|------|-----------|
| 强调重点 | `⚠️ ✅ ❌ 💡 📌` |
| 列表序号 | `1️⃣ 2️⃣ 3️⃣` 或 `🔸 🔹 🔸` |
| 情感表达 | `💕 💝 ❤️ 🥺 😭` |
| 互动引导 | `💬 👇 🔄` |
| 分类标签 | `📚 💰 🎯 🌟 ⭐` |

## 互动话术库

| 类型 | 话术模板 |
|------|---------|
| 提问式 | `你最想试哪个？评论区告诉我！` |
| 共情式 | `有没有和我一样的姐妹？举个手🙋` |
| 收藏式 | `建议收藏，以后一定用得上！` |
| 分享式 | `转发给你最好的朋友！` |
| 补充式 | `还有什么想补充的？评论区见！` |
